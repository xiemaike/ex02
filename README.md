# BlockHaven · 方块岛

BlockHaven 是一款原创浏览器方块沙盒游戏，包含程序化岛屿、采集建造、体素角色、怪物战斗、昼夜变化和本地存档。游戏使用 Three.js 与 Vite 开发，没有使用《Minecraft》的官方贴图、模型或商标资源。

## 立即游玩

打开在线版本：<https://xiemaike.github.io/ex02/>

推荐环境：桌面版 Chrome 或 Edge、键盘和鼠标。当前版本以电脑操作为主，手机触控尚未完整适配。

### 第一次进入

1. 点击首页的“进入世界”。
2. 选择一个方块角色预设，也可以修改昵称、肤色、上衣和发色。
3. 点击“保存角色并出发”。
4. 浏览器会请求锁定鼠标；锁定后移动鼠标即可转动视角。
5. 按 `Esc` 可释放鼠标并打开暂停菜单。

如果第一次不知道如何操作，可以点击首页或暂停菜单中的“玩法说明”。

首页和暂停菜单还提供“角色展示台”：无需进入生存玩法即可切换全部角色，并预览待机、行走、奔跑、跳跃、攻击和通话动作。每种动作都带有独立的方块舞台特效，包括呼吸光环、落脚方块尘、速度残影、起跳冲击环、体素斩击和通讯波纹。

## 完整操作

| 按键或操作 | 功能 | 补充说明 |
| --- | --- | --- |
| `W A S D` | 前后左右移动 | 移动方向跟随当前视角 |
| `Shift` | 冲刺 | 按住时提高移动速度 |
| `Space` | 跳跃 | 落地后才能再次跳跃 |
| 移动鼠标 | 转动视角 | 需要先点击游戏画面锁定鼠标 |
| 鼠标左键 | 攻击或破坏 | 准星对准怪物时攻击，否则破坏方块 |
| 鼠标右键 | 放置方块 | 放置当前快捷栏选中的方块 |
| `1`–`6` | 选择方块 | 对应快捷栏六个位置 |
| 鼠标滚轮 | 切换方块 | 向前或向后循环选择 |
| `C` | 打开角色编辑器 | 可更换角色、昵称与颜色 |
| `F5` | 切换视角 | 第一人称 → 背面第三人称 → 正面展示 |
| `Esc` | 暂停 | 释放鼠标并显示暂停菜单 |

## 采集与建造

屏幕中央的十字是准星。将准星对准距离足够近的方块：

- 左键破坏方块；世界最底层不可破坏。
- 右键在目标方块相邻位置放置新方块。
- 如果新方块会与玩家身体重叠，则不会放置。
- 快捷栏提供草地、泥土、岩石、原木、树叶和沙砾六种方块。
- 左下角会显示坐标和累计放置数量。

## 怪物与战斗

岛上会生成四类怪物：

| 怪物 | 特点 | 建议 |
| --- | --- | --- |
| 苔原史莱姆 | 生命较低，会弹跳接近玩家 | 适合熟悉战斗操作 |
| 暗影兽 | 移动速度快 | 保持距离并连续攻击 |
| 石像守卫 | 生命和伤害最高 | 不要被它逼到地形边缘 |
| 火焰精灵 | 悬浮移动 | 注意它从不同高度靠近 |

准星瞄准怪物时会变红，此时点击左键进行近战攻击。怪物靠近后会伤害玩家；生命归零时，玩家会回到出生点并恢复生命。击败怪物会掉落发光晶体，靠近即可自动拾取。夜间怪物的追击能力更强。

## 角色与视角

按 `C` 打开角色编辑器。目前包括：

- 薄荷通讯者：专属像素脸、分段双辫、手机和通话姿势。
- 水之歌者：白蓝长发、海蓝礼服和方块礼帽。
- 樱花魔法使：粉色方块双马尾与蝴蝶结。
- 金焰骑士：金发、王冠、肩甲和分层衣摆。
- 月夜巫女：黑色分块长发与红白巫女服。
- 森灵弓手：绿色短发与方块精灵耳。

`F5` 会循环切换三种视角。正面展示用于查看角色脸部、服装和配件；该视角主要用于观察模型，正常探索时建议使用第一或背面第三人称。

## 保存与重新开始

- 方块修改、角色、位置、视角、生命和晶体会保存在当前浏览器的 `localStorage` 中。
- 破坏或放置方块后会自动保存，也可点击顶部“保存”。
- 刷新或关闭页面后，再次使用同一浏览器打开即可继续。
- 暂停菜单中的“生成新世界”会清除当前建造、晶体和怪物状态，但保留角色设置。
- 清理浏览器站点数据、使用无痕模式或更换浏览器后，旧存档不会自动迁移。

## Windows 本地运行

### 1. 安装环境

安装 [Node.js 22 LTS](https://nodejs.org/)。安装完成后打开命令提示符，检查版本：

```bat
node --version
npm --version
```

如果提示“不是内部或外部命令”，请重新打开命令提示符；仍然无效时重新安装 Node.js，并确保安装程序启用了添加到 `PATH` 的选项。

### 2. 获取项目

方法一：在 GitHub 页面点击 **Code → Download ZIP**，解压后进入项目文件夹。

方法二：已经安装 Git 时执行：

```bat
git clone https://github.com/xiemaike/ex02.git
cd ex02
```

### 3. 安装依赖并启动

在项目目录的命令提示符中执行：

```bat
npm install
npm run dev
```

终端会显示类似 `http://localhost:5173/` 的地址。按住 `Ctrl` 点击地址，或复制到浏览器打开。开发服务器运行期间不要关闭该终端；需要停止时按 `Ctrl+C`。

## 构建生产版本

```bat
npm run build
```

构建结果会生成在 `dist` 文件夹中。检查生产版本：

```bat
npm run preview
```

不要直接双击 `dist/index.html`，浏览器的本地文件限制可能导致脚本无法加载；应使用 `npm run preview` 或其他静态服务器。

## 常见问题

### 页面打开后不能转动视角

点击游戏画面，让浏览器进入鼠标锁定状态。如果浏览器弹出权限提示，请选择允许。按 `Esc` 会退出锁定，再次点击画面即可继续。

### `npm install` 下载失败

确认网络能够访问 npm，并尝试：

```bat
npm cache clean --force
npm install
```

### 在线页面显示旧版本

GitHub Pages 更新可能需要几分钟。等待 Actions 部署完成后，使用 `Ctrl+F5` 强制刷新浏览器缓存。

### 画面卡顿

关闭其他占用显卡的程序，降低浏览器缩放或窗口大小，并关闭浏览器节能模式。集成显卡设备建议不要同时开启多个游戏页面。

### 想彻底删除存档

打开浏览器开发者工具，在 Application/应用 → Local Storage 中删除 `blockhaven-save-v1`；也可以清除该网站的站点数据。
