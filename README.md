# BlockHaven · 方块岛

一款原创的浏览器方块沙盒小游戏。它采用程序化地形，无需下载客户端，打开网页即可采集、搭建和探索。

## 已实现

- 第一人称移动、冲刺和跳跃
- 破坏与放置 6 种方块
- 程序化岛屿、海水、沙滩与树木
- 动态昼夜和实时光影
- 准星选中反馈与快捷栏
- 浏览器本地自动存档
- 响应式游戏 HUD

## 操作

| 按键 | 功能 |
| --- | --- |
| `W A S D` | 移动 |
| `Shift` | 冲刺 |
| `Space` | 跳跃 |
| 鼠标左键 | 破坏方块 |
| 鼠标右键 | 放置方块 |
| `1`–`6` / 滚轮 | 切换方块 |
| `Esc` | 暂停 |

## 本地运行

需要 Node.js 20.19+ 或 22.12+。

```bash
npm install
npm run dev
```

生产构建：

```bash
npm run build
```

## 技术栈

Three.js、Vite、原生 JavaScript/CSS。所有方块纹理由代码即时生成，没有使用《Minecraft》的素材或商标资源。
