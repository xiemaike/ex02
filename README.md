# BlockHaven · 方块岛

一款原创的浏览器方块沙盒小游戏。它采用程序化地形，无需下载客户端，打开网页即可采集、搭建和探索。

## 已实现

- 第一人称移动、冲刺和跳跃
- 破坏与放置 6 种方块
- 程序化岛屿、海水、沙滩与树木
- 动态昼夜和实时光影
- 可自定义昵称、肤色、服装与发色的方块角色
- 芙宁娜风格“水之歌者”与 4 套原创动漫职业预设
- “薄荷通讯者”专属高精度体素模型：像素脸、分段双辫、分层服装、手机与通话姿势
- `F5` 可循环切换第一人称、背面第三人称和正面角色展示
- 史莱姆、暗影兽、石像守卫、火焰精灵四类怪物
- 怪物追击 AI、生命值、近战战斗、血条与晶体掉落
- `F5` 第一/第三人称即时切换
- 准星选中反馈与快捷栏
- 浏览器本地自动存档
- 响应式游戏 HUD

## 操作

| 按键 | 功能 |
| --- | --- |
| `W A S D` | 移动 |
| `Shift` | 冲刺 |
| `Space` | 跳跃 |
| 鼠标左键 | 破坏方块 |
| 鼠标右键 | 放置方块 |
| `1`–`6` / 滚轮 | 切换方块 |
| `F5` | 切换第一/第三人称 |
| `C` | 打开角色编辑器 |
| `Esc` | 暂停 |

## 本地运行

需要 Node.js 20.19+ 或 22.12+。

```bash
npm install
npm run dev
```

生产构建：

```bash
npm run build
```

## 技术栈

Three.js、Vite、原生 JavaScript/CSS。所有方块纹理由代码即时生成，没有使用《Minecraft》的素材或商标资源。
